# Data-Analytics-Project-Customer-Insights-using-Statistical-Analysis-Dashboarding
📊 Customer Insights Analysis using Statistical Methods & Hypothesis Testing

This project focuses on analyzing customer data to uncover meaningful patterns and validate business assumptions using statistical techniques. The goal is not just to explore the data, but to support insights with proper hypothesis testing and data-driven reasoning.

🎯 Objective

Organizations often make decisions based on assumptions about customer behavior. In this project, I analyzed customer data to identify trends and statistically test whether those assumptions are valid.

🔍 Key Steps Performed
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Statistical Analysis (Mean, Median, Mode)
Correlation Analysis
Hypothesis Testing
Data Visualization for insights
🛠️ Tools & Technologies
Python
Pandas
NumPy
Matplotlib
Seaborn
Statistics Concepts
Descriptive Statistics (Mean, Median, Mode)
Correlation Analysis
Hypothesis Testing
📈 Outcome
Identified key customer behavior patterns
Validated insights using statistical methods
Created visualizations to clearly communicate findings
Enabled data-driven decision-making
